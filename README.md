# ByteBank
Um projeto desenvolvido no aprendizado de C# e Orientação a Objetos, o ByteBank é um banco fictício onde foi implementado e consolidado todos os conhecimentos adquiridos durante a formação: 

* Como criar minhas próprias classes, os conceitos de herança e interface;
* Aprendi sobre exceções, criando meu próprio tipo de exceção;
* Aprendi a criar minha própria biblioteca de modelos e a utilizar o NuGet;
* Entendi como funcionam arrays e tipos genéricos;
* E também compreendi entrada e saída com streams.
